Intro to Github & Command Line

By Katie Kowalsky

Goals for this tutorial:

This tutorial serves as a guide to learning the basics of Github and running version control software from the command line. Numerous chapters and activities serve to help you in learning this software that is often stressful and confusing to many beginner programmers.My biggest piece of advice for tackling the learning curve is to just study this. Write down the commands on notecards and spend 10 minutes a day reviewing the basic commands.
